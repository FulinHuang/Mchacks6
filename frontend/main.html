Challenge Description


 The challenge is to use radio telemetry to identify if a wireless connection is adequate or if a new connection must be sought out. Using pre-rendered data, you can interpolate and make better decisions while travelling.

Show telemetry for visual inspection. Use the user interface to visualize these values. (ex: Google Map API, Gnu plot, others)
